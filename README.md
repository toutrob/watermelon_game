<h1>Watermelon Game</h1><p>Watermelon Game est une variante du célèbre jeu Suika Game, mais avec des planètes et dans l'espace. Le gameplay est similaire à l'original avec quelques fonctionnalités supplémentaires qui ajoutent de la profondeur au jeu.</p><h2>Fonctionnalités</h2><ul><li><strong>Gameplay classique de Suika Game</strong> : Combinez des planètes pour créer des planètes plus grandes et gagner des points.</li><li><strong>Deux pouvoirs spéciaux</strong> :<ul><li><strong>Inverser la gravité</strong> : Permet d'inverser temporairement la gravité pour remuer les planètes dans le jeu.</li><li><strong>Supprimer une planète</strong> : Permet de supprimer une planète au choix.</li></ul></li></ul><h2>Captures d'écran</h2>
